Mac的终端下如何快速前进后退一个单词

***TLDR: ***

Alt/Option + B 是后退一个单词Backward。

Alt/Option + F 是前进一个单词Forward

在终端下，一直会用的几招是

  • Ctrl + A 回到命令的最开始
  • Ctrl + E 回到命令的末尾
  • Ctrl + U 是清空当前命令

可是很多时候也需要快速的向光标右侧移动。比如下方这样，命令输入了一半，出现了比较符合的预测命令。

这个时候我需要能快速的基于推测的命令输入 git commit --fixup，然后手动调整后面的参数。

经常遇到这样的情况，只能快速反复的按右箭头。一直按都不行，响应慢。

今天是终于受不了了，耐心Google了一下，终于找到Mac下合适的快捷键

  • Alt/Option + B 是后退一个单词Backward
  • Alt/Option + F 是前进一个单词Forward
